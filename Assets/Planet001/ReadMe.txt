mark@secondmousestudios.com

=====================================================================

Description
--------------------
This package includes 88 terestrial planet materials. The planet material dimensions are 1440x720.

They represent the same planet in 8 seasonal positions: 
  Spring
  Spring/Summer
  Summer
  Summer/Fall
  Fall
  Fall/Winter
  Winter
  Winter/Spring

They also represent the same planet in land percentage from 0% land to 100% land in increments of 10.

There are 88 sample spheres in the project that each have a map of each material attached as well as
a rotate script so you can see them from all angles. The camera also has a rotate script so you can 
see all of the planets froma closer vantage point.

Refer to the nameof the PNG to determine the planets properties. For example:

The PNG: [22_020PercentLand,6Fall-Winter.png] tells you the number of the PNG file and the
associated planet (22).  The next three numbers represent the percentage of land (20%).  The next
number represents the season (6 - Fall/Winter).
